# Movie Booking System

A web-based Movie Booking System built using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap.

## Features

- User Registration & Login
- Movie Listings
- Movie Details Page
- Ticket Booking
- Booking Management
- User Dashboard
- Admin Panel
- Manage Movies
- Manage Bookings
- Responsive Design

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap

## Installation

1. Clone the repository

```bash
git clone https://github.com/NikeshTarur0/movie-booking-system..git
```

2. Import `database.sql` into MySQL.

3. Configure database settings in `config.php`.

4. Start Apache and MySQL using XAMPP.

5. Open:

```
http://localhost/movie-booking-system
```
